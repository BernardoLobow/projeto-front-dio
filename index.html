O que é Semântica

No português, semântica se refere a área que estuda o significado de palavras frases e afins. No entanto, na programação refere-se ao significado do trecho de código, por

por exemplo, qual a finalidade ou função aquela linha de código tem. A semântica é muito importante e necessária, pois diz claramente o que aquele elemento significa tanto

para o programador, quanto para os navegadores.

Elementos semânticos

A seguir veremos exemplos de elementos semântico:

  • Mark: Funciona como um marca texto
    Exemplo mark
  • Blockquote: Indica uma longa citação, podendo adicionar a url da fonte.

    Exemplo:

    "Um ser humano deve transformar informação em inteligência ou conhecimento. Tendemos a esquecer que nenhum computador jamais fará uma nova pergunta."

  • Del: representa a parte do texto que foi exclída do documento.

    Exemplo del

  • i: Antes usada como itálico, porém agora é utilizada pra destacar algo, podendo ser termos técnicos, outros idiomas e etc.

    Exemplo: A expressão Carpe diem é muito utilizada hoje em dia e significa aproveite o momento/dia

  • u: Essa tag é utilizada para indicar erros de português ou para sublinhar trechos específicos sem indicar ênfase forte

    Exemplo: A linguagem HPML é a base da internet

  • hr: Representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção).

    Exemplo: Primeiro paragrafo do conteúdo.

    Segundo paragrafo do conteúdo.

  • sup: Utilizado para definir texto sobrescrito.

    Exemplo: x2+2x=0

  • sub: Utilizado para definir texto subscrito.

    Exemplo: A molécula O2 é imprescindível para a vida, bem como o H2O.
