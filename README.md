# Importância da Semântica no HTML 🏷️

Este projeto foi desenvolvido como parte de um curso da **DIO (Digital Innovation One)**. O objetivo principal foi explorar e demonstrar como a semântica no HTML5 influencia tanto a legibilidade do código para desenvolvedores quanto a interpretação pelos navegadores e motores de busca.

## 🚀 Tecnologias Utilizadas
- HTML5 (Estruturação semântica)

## 🧠 Conceitos Praticados
- **Elementos de Estrutura:** Uso de `<main>`, `<article>` e `<footer>`.
- **Hierarquia de Títulos:** Aplicação de `<h1>` a `<h6>`.
- **Tags de Formatação Semântica:** Uso de `<mark>`, `<del>`, `<code>`, `<sup>` e `<sub>`.
- **Acessibilidade:** Uso de atributos como `cite` em blockquotes e `abbr` para siglas.
- **Navegação Interna:** Criação de âncoras para facilitar a navegação na página.

## 📄 Como visualizar
Como este é um projeto de "HTML cru", basta baixar o arquivo `index.html` e abri-lo em qualquer navegador.
